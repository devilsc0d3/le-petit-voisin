# Website LE PETIT VOISIN
Welcome to the repository for the website of **Le Petit Voisin**, a cozy bar located in the heart of Toulouse, France. This site showcases their creative cocktails, local beers, live music nights, and warm atmosphere.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- JSON for internationalization (i18n)
- Responsive design principles

## Features
- Multi-language support (French and English)
- Dynamic content sections (About, Cocktails, Happy Hour, Location, Contact)
- Custom styling with CSS for a unique look and feel
- Interactive elements for enhanced user experience
